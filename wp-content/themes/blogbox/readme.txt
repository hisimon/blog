colors, text colors, font styles, static pages, sidebars, captcha for comments and contact page.

== Description ==

	blogBox is a full featured responsive WordPress theme offering the user many options to 
	control the look of the theme, and many options for special pages and content. 

	1) Home Page
	    - full width feature slider, button or thumbnail navigation
	    - small feature slider with a text area, buttons or thumbnail navigation, 
	      complete with an option to widgetize the text area
	    - single image options
	    - two slogan areas
	    - service area with three sections where you can post an image, a title,
	      and some text.
	    - select a home blog template with all the above features and your blog posts.

	2) Header Options
	    - header image, logo or full width banner
	    - widgetized area for social links, phone number
	    - optional display for blog title and description
	    - Menu can be positioned left, right, or center
	    
	3) Footer Options
	    - Option to exclude footer
		- footer can be 3 or 4 columns
		- Option to exclude copyright strip
		- Copywrite area with editable html text left, center, and right

    4) Post Options
	    - Post Formats - All post formats are available
		- Post Format Helper Plugin - Separate post format plugin is available 
		  for Audio, Quote, and Video post formats
		- Display single posts with a sidebar or fll width 
		- Optionally display Timestamp, Author, Categories, and Tags
		
	5) Custom Styling
	    - Background Colors - change 16 different background colors
		- Text Colors - change 17 different text colors
		- Post Formats - change background colors and text colors for Aside, Audio, Chat, 
			Link, Status, and Quote post formats
		- Fonts - choose from 9 standard fonts and 13 Google Fonts for Headers and Body
		- Menu - menu can be styled separately from header.
		- Skins - use preset skins if you don't want to select your own colors
		
	6) Social Widget 
		- delicious, digg, facebook, google+, linkedin, pinterest, myspace, rss, tumblr,twitter
		- 24x24px or 32x32px
		- select only the social links you want for each widget instance

	7) Templates
	    - full width
	    - full width blog
	    - Contact template with custom captcha and widgetized sidebar
	    - Siteplan template
	    - two left sidebar templates - with separate widgetized sidebar
	    - two right sidebar template - with separate widgetized sidebar
	    - Archive template
	    - up to 5 different Portfolio Templates, that
	      you can customize with up to 4 columns, c/w post content
	      or feature media text options
	     
	8) Other Features
	    - Fully Responsive, user can optionally disable this feature
		- Colorbox - Display galleries and photos using Colorbox.
		- Comment Captcha
		- Color Captcha or black and white
		- Font Awesome - great vector font icon package now part of blogBox
		- PageNavi ready - download the PageNavi plugin for great page 
		  navigation features.
		- Child Theme Ready

== Installation ==

1) Upload to the themes directory of your wordpress setup. Ensure all sub directories are maintained.

2) Make sure you install only the parent folder of the blogBox files....meaning wp-content/themes/blogBox/...files. 
   There should not be 2 blogBox folders....ie wp-content/themes/blogBox/blogBox/....files.
3) The alternative is to go to your WordPress installation. Go to Appearance => Themes and select the Install tab. 
   You can search for the blogBox Theme and install it from there.</li>
4) Activate the theme through the Wordpress Admin panel under "Appearance" and "themes"
